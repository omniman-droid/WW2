# Operation Iron Wolf: 1944

A retro corridor shooter inspired by classic Wolfenstein-style gameplay, themed as **Allied Rangers vs Axis bunker garrison**.

## Play locally with Node.js

```bash
npm start
```

Then open: `http://localhost:3000`

Shareable play link format (auto-start mission):

- `http://localhost:3000/?join=1`

## Deploy on Netlify

- Connect this repo in Netlify.
- Publish directory is project root (`.`).
- `netlify.toml` already routes `/` to `Index.html`.

## Controls

- **Move:** W/A/S/D
- **Strafe:** Q
- **Shoot:** E
- **Aim:** Mouse (click game window for pointer lock)
- **Fire:** Left click (or Space)
- **Reload:** R
- **Start / Redeploy:** Enter
